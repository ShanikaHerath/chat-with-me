# Chat With Me 💬 (Lakma AI)

A modern, responsive, and feature-rich AI chat application. This project features a React-based frontend and a Node.js/Express backend, allowing users to interact with powerful AI models like Google's Gemini API and Hugging Face.

## ✨ Features

- **🤖 AI Model Integration**: Chat seamlessly with advanced AI models powered by Gemini API.
- **🎨 Modern UI/UX**: A sleek, glassmorphic chat interface built with React and styled with custom CSS.
- **🌗 Theme Toggle**: Easily switch between Light and Dark modes.
- **⚙️ Model Selector**: Switch between different AI models on the fly directly from the header.
- **👤 Profile Management**: Customizable user profile settings with avatar support.
- **📱 Responsive Design**: Fully responsive chat interface that works beautifully on mobile and desktop devices.

## 🛠️ Tech Stack

**Frontend:**
- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Lucide React](https://lucide.dev/) (Icons)
- [React Markdown](https://remarkjs.github.io/react-markdown/) (For formatting AI responses)

**Backend:**
- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [dotenv](https://www.npmjs.com/package/dotenv) (Environment variables)
- [CORS](https://www.npmjs.com/package/cors)
